Kinolu Color Transfer (Demo)

Reference -> Target, 2D strength, optional micro edits. Calls POST /api/transfer

Inputs

2D Strength Pad

Drag in the square or type values. X=Color, Y=Tone.

RGB Curves (可拖拽) / 曲线

Click line to add a point, drag to move, double-click point to remove. Endpoints stay fixed.

Micro Edits (optional) / 基础调色

HSL 7-way (Lightroom style)

If you cannot see these controls, hard refresh the page (Cmd+Shift+R).
